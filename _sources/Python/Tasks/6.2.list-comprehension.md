# Генераторы списков

## Список с алфавитом

Сгенерировать список, который состоит из алфавита. Ожидаемый результат: 

```
['а', 'б', 'в', 'г', 'д', 'е', 'ж', 'з', 'и', 'й', 'к', 'л',
'м', 'н', 'о', 'п', 'р', 'с', 'т', 'у', 'ф', 'х', 'ц', 'ч',
'ш', 'щ', 'ъ', 'ы', 'ь', 'э', 'ю', 'я']
```

<details><summary>Решение</summary>
<pre>
A = [chr(x) for x in range(1072, 1103+1)]
</pre>
</details>
